# Deafness in Brazil
Analysis of comments of a "Manual do mundo" video to extract info about deafness in Brazil

Me with the great help of @IanFPFerreira plotted graphics of the comments of the video using libraries as Pandasm, Seborn and NumPy

This project is a son of other project mine, the repository is https://github.com/italovinicius18/armazenador-de-comentarios

Source video comments = https://www.youtube.com/watch?v=uoxtAXN2tnI
